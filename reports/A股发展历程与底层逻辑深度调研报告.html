1. A股的历史发展阶段

1.1 总体脉络

中国A股市场从1990年诞生至今，经历了一条**"从无到有、从小到大、从封闭到开放、从审批到注册"的演进路径。中国人民大学吴晓求教授将这一历程概括为"三座丰碑"**：沪深交易所创设（第一座）、股权分置改革（第二座）、注册制改革（第三座）。

1.2 五个发展阶段

阶段一：萌芽起步期（1990—1998）——"姓资姓社"的争论中诞生

阶段特征：在意识形态争论中艰难起步，市场规模极小，制度几乎空白，处于"试点"状态。

时间 关键事件 深层意义
1990年12月 上交所（12.19）、深交所（12.1试营业）开业，首批仅13只股票（沪市"老八股"：飞乐音响、爱使电子、申华电工等；深市"老五股"：深发展、深万科等） 邓小平"证券、股市要坚决地试"的政治背书下的产物
1992年1-2月 邓小平南巡讲话，明确"证券、股市这些东西究竟好不好……要坚决地试" 终结"姓资姓社"争论，为资本市场正名
1992年5月 上证指数单日暴涨105%，"8·10"深圳新股认购事件 暴露出制度严重滞后于市场需求——排队三天三夜抢购认购表
1992年10月 国务院证券委员会和证监会成立 统一的监管体系从无到有，刘鸿儒任首任主席
1993年 股票发行实行审批制，"额度管理"阶段——国务院每年下达总发行额度，分配给各省市 "跑部钱进"的根源——上市指标是稀缺资源
1996年 "十二道金牌"事件——证监会连续发文打压股市过热，上证从1258跌至855点 A股"政策市"基因的早期显现
1998年4月 国务院机构改革，证券委与证监会合并，集中统一监管体系形成 监管权力由分散走向集中
1999年7月 《证券法》正式实施 证券市场终于有了"基本法"——此前主要靠行政规章运行

阶段一的关键数字：

  • 1990年总市值：约10亿元人民币（不到单只美股中等市值）
  • 1998年总市值：约1.95万亿元（8年增长近2000倍，主要是扩容而非股价上涨）
  • 1999年上市公司：约950家
  • 投资者开户数：从零增长至约4000万户

制度基因：审批制+额度管理奠定了此后二十多年A股"壳价值"的制度基础——上市资格本身成为一种稀缺资产。这一基因至今仍在影响A股定价（ST股壳价值、借壳上市等）。

阶段二：规范发展期（1999—2004）——从"无序"走向"有序"

阶段特征：在"5·19"行情中开启，在四年熊市中沉淀，在"国九条"中明确方向。这是A股从野蛮生长到制度化治理的转折期。

时间 关键事件 深层意义
1999年5月 "5·19"行情爆发——上证从1047点飙升至1756点（30个交易日涨67%），科技网络股狂潮 人民日报发表《坚定信心，规范发展》社论背书做多——"政策造牛"的经典案例
2000年3月 发行制度从审批制过渡到核准制（通道制→保荐制，标志：取消额度管理） 首次将上市判断权从政府转移到中介机构
2001年6月 国务院发布《减持国有股筹集社会保障资金管理暂行办法》 引发市场恐慌——国有股按市价减持意味着天量抛压
2001年10月 证监会紧急叫停国有股减持 "纠错但已崩"——上证已从2245跌至1514点，宣告四年熊市开始
2002年12月 QFII制度正式引入，瑞银华宝为首家 A股对外开放的第一步，但初期额度极其有限
2003年10月 《证券投资基金法》通过 公募基金行业从此有法可依，为机构化铺路
2004年1月 国务院出台第一个"国九条"——《关于推进资本市场改革开放和稳定发展的若干意见》 首次将资本市场发展上升到国家战略层面，但具体路径尚未明确；首次提出"积极稳妥解决股权分置问题"
2004年5月 深交所设立中小板，首批8家 为民营企业、中小企业开辟融资渠道，也是创业板"试验田"
2004年全年 上证从1783点跌至1266点 经历股权分置焦虑下的漫长探底，市场信心极度脆弱

阶段二的关键背景：股权分置——A股的"原罪"

在股权分置改革前（1990-2005），A股上市公司的股权结构是分裂的：

┌──────────────────────────────────────────────────┐
│              上市公司总股本                        │
├────────────────────┬─────────────────────────────┤
│   流通股 (~1/3)    │     非流通股 (~2/3)          │
│   可在二级市场交易  │   不可流通（国有股、法人股）    │
│   投资者用现金申购   │   按净资产折股（成本极低）     │
│   大股东不持有       │   大股东全部持有              │
└────────────────────┴─────────────────────────────┘

后果：
1. 大股东利益与股价无关 → 无心提升股价 → 热衷掏空上市公司
2. 2/3的股份不能流通 → 市场定价机制失灵 → 小盘股操纵盛行
3. 非流通股若按市价减持 → 对二级市场形成巨大抛压预期
4. 这是2001-2005四年熊市的制度根源

阶段二的关键数字：

  • 上证指数区间：1047（"5·19"起点）→ 2245（2001年顶）→ 998（2005年股改前底）
  • 最大回撤：55.5%（2245→998，历时约4年）
  • QFII首批额度：仅约40亿美元（vs 目前超1000亿美元）

阶段三：重大制度变革期（2005—2008）——"股改牛"与制度套利

阶段特征：A股历史上最重要的制度变革——股权分置改革，催生了迄今为止最大的牛市（6124点），也暴露了全流通时代的新风险。

时间 关键事件 深层意义
2005年4月29日 证监会发布《关于上市公司股权分置改革试点有关问题的通知》，股改正式启动 三一重工等4家公司首批试点
2005年6月 上证触底998.23点后开启反转 "股改底"——送股对价让流通股东获得约30%的额外股份
2005年9月 股改全面推进，《上市公司股权分置改革管理办法》出台 对价方案：每10股流通股获送约3股
2006年 上证全年涨130%——"股改对价+盈利增长+人民币升值"三重共振 公募基金规模爆发，散户跑步入场
2007年5月 "5·30"半夜上调印花税（0.1%→0.3%），上证5天跌21% A股"政策调控"的典型操作——用税收工具直接干预市场
2007年8月 美国次贷危机蔓延，但A股仍在惯性冲高 全球危机与A股狂热的背离
2007年10月16日 上证触及6124.04点历史峰值 静态PE超70倍，PB超7倍——估值已完全脱离基本面
2007年11月 中国石油上市（首日48.6元），随后暴跌80%+ "问君能有几多愁，恰似满仓中石油"——48元买入者15年后仍未解套
2007年底 股改基本完成：1333家公司完成股改，占比98% A股正式进入全流通时代
2008年 上证从5261跌至1820点（全年跌65.4%），史上最大年度跌幅 次贷危机+大小非解禁+估值泡沫三重打击

股权分置改革的核心机制——"对价"

股改的交易结构：
非流通股股东向流通股股东支付"对价"（通常为送股）
以换取非流通股的"流通权"

典型案例：三一重工
方案：每10股流通股获送3.5股 + 8元现金
结果：流通股东获得约35%的额外股份作为补偿

为何要送股？——非流通股原本按净资产折股（成本远低于流通股东）。
若直接全流通而不补偿，流通股东将面临天量低成本筹码的抛售。

全体流通股东获得的"对价红包"约为总市值的30%——这直接构成了2005-2007年大牛市的一部分收益。

全流通后的新问题——"大小非解禁"：

全流通 ≠ 全部立即流通。股改设置了限售期：

"大非"（持股5%以上）：股改完成36个月后可逐步减持
"小非"（持股5%以下）：股改完成12个月后可逐步减持

2008年是"大小非"解禁的第一个高峰期
→ 产业资本首次获得在二级市场减持套现的通道
→ 大量持股成本接近于零的筹码涌入市场
→ 这是2008年A股跌得比全球金融危机中其他市场更惨的制度性原因之一

从此以后，"减持"成为A股永恒的敏感词——每一轮反弹都面临产业资本减持的压力。

2.2 盈亏极化的结构性不平等——"散户绞肉机"

这是A股最残酷但最真实的数据画像：

账户规模 盈利比例 亏损比例 含义
10万元以下账户 仅2% 98% 微型散户几乎系统性亏损
10-50万元 \~15-20% \~80-85% 中小散户盈亏比严重失衡

为什么会出现这种极端分化？——五重结构性劣势

1. 信息劣势
   机构：调研预约+卖方路演+产业链验证+卫星数据
   散户：F10公告（往往滞后3-6个月）+ 股吧传闻

2. 工具劣势
   机构：股指期货对冲+期权+融券做空+大宗交易折价买入
   散户：只能做多，T+1当天买入被套无法止损

3. 速度劣势
   量化机构：微秒级（千分之一毫秒）信号+执行
   散户：至少300毫秒延迟 → 股价已波动2%-5%

4. 认知劣势
   机构：研究团队覆盖+财务模型+行业数据库
   散户：看K线+追涨停+听大V推荐

5. 行为劣势
   散户年换手率>500%（机构5-10倍）
   频繁交易 → 印花税+佣金吞噬本金3%-5%/年
   涨时贪婪不卖、跌时恐慌割肉——经典"高买低卖"

散户行为的心理学基础：三个核心偏误

偏误 表现 在A股的具体体现
过度交易偏误 过度自信导致频繁交易 散户年换手500%+，每2-3个月清仓一次，"不交易就难受"
处置效应 赚一点就卖，亏了死扛 盈利股平均持有15天，亏损股平均持有90天——"截断利润，让亏损奔跑"
羊群效应 跟风追涨，恐慌割肉 涨停板次日散户净买入最多（追板），跌停板散户净卖出最多（踩踏）

核心数据：小资金投资者平均持股周期约30天（美股约300天），高频交易加之高昂的隐性成本（印花税0.1%+佣金+冲击成本），年化交易费用可吞噬小资金本金的3%-5%。在长期"七亏二平一赚"的市场中，光是交易成本就足以让散户跑输。

T+1的"保护错觉"：

维度 设计初衷 实际效果
抑制投机 阻止当日频繁买卖，减少过度投机 散户确实无法频繁交易，但机构/游资通过底仓滚动做T+0

2.4 深层制度性特征——六层"不对称"

A股的结构性不对称（每一层都使散户处于劣势）:

第1层：方向不对称
    ←→ 做多（所有人）+ 做空（仅有融券渠道，散户几乎用不到）
    → 天然的多头市场 → 泡沫容易膨胀，暴跌时没有反向力量缓冲

第2层：时间不对称
    T+1（散户）vs 变相T+0（机构/游资）
    → 散户持仓被锁定一天，机构可当日滚动套利

第3层：价格不对称
    涨跌停 vs 机构大宗交易折价买入
    → 机构以9折拿到筹码，散户在涨停板上追

第4层：信息不对称
    机构提前调研+卫星数据+产业链验证 vs 散户看公告
    → 财报公布前机构已布局/撤退完毕

第5层：速度不对称
    量化微秒级 vs 散户300ms+
    → 散户看到的行情已是300ms前的，"别人已经买完了你才看到"

第6层：成本不对称
    机构：印花税+极低佣金（万1以下）+算法交易降低冲击成本

3.2 维度一：政策驱动——A股最独特的定价因子

核心命题：在A股，政策的边际变化直接改变市场对长期确定性的定价（即风险溢价ERP），其影响力远超当期盈利变化。

政策如何影响ERP？——传导机制

政策信号 → 改变长期确定性预期 → ERP上升/下降 → 估值中枢移动

正例（ERP下降，估值上移）：
  科创板设立(2019) → "科技自立"有了制度保障 → 半导体估值体系重塑
  央企市值管理(2024) → 央企治理改善预期 → 中字头PB修复（0.5→0.8+）
  第三个国九条(2024) → "投资市"定位 → 中长期资金愿意给更高估值

反例（ERP上升，估值下移）：
  2021年"防止资本无序扩张" → 互联网/教育/地产不确定性飙升 → 暴跌
  2018年去杠杆+贸易战 → 经济增长不确定性 → 全年跌25%
  2015年清理配资 → 市场制度不确定性 → 千股跌停踩踏

3.3 维度二：流动性驱动——A股的"水龙头"

核心命题：A股的短期（1-2年）估值波动几乎完全由流动性周期驱动。在A股，"放水"≈"涨"，"收水"≈"跌"，这一规律比美股更直接、更强。

流动性对A股估值的传导——"剩余流动性"框架

剩余流动性 = M2增速 - 社会融资规模增速

当剩余流动性↑（M2增速 > 社融增速）：
  实体信贷需求弱 + 货币宽松 → "钱多但无处可去" → 涌入金融市场
  → 对利率敏感的成长股/小盘股跑赢
  → A股历史上每一次大行情都伴随着剩余流动性扩张

当剩余流动性↓（M2增速 < 社融增速）：
  实体信贷需求旺盛 + 货币收紧 → "钱去了实体经济"
  → 股市资金被抽走 → 整体承压，仅剩结构性机会

"从海外定价到国内定价"的切换

2021-2023年：海外定价主导
  中美利差倒挂 → 外资流出 → A股受美债利率攀升高压
  → 即使国内降息，估值也难以扩张
  → "美债利率是A股的隐形天花板"

2024年起：国内定价切换
  美债利率仍高，但国内降息力度更大且社融修复
  → 外资流出压力减轻，国内资金成边际定价主力
  → 估值修复由国内利率下行驱动，而非美债
  → "内外定价权重正在发生历史性切换"

3.4 维度三：基本面驱动——从"当期盈利"到"永续增长预期"

核心命题：A股对"远期故事"的定价权重远超当期盈利——牛市顶部往往以极度乐观的g预期为支撑，熊市底部则以g崩溃为特征。

基本面定价的"两步走"框架

阶段一：先涨估值（预期驱动）
  "我预期未来盈利会好" → g*预期上升 → 估值扩张
  即使当期EPS还没动，PE已经从10倍涨到20倍
  → 牛市初期的典型特征

阶段二：再涨盈利（现实验证）
  盈利确实兑现了 → EPS上升 → 即使PE不再扩张甚至收缩
  股价的上涨由盈利增长接棒
  → 牛市中后期的典型特征

阶段三：戴维斯双杀（崩溃）
  盈利不及预期 + g*预期崩塌 → PE和EPS同时收缩
  → 熊市的典型特征

  买入 → 可即时卖出
  → 日内交易者可随时调整
  → 错误定价可被当日纠正（套利力量即时进场）
  → 不存在"被锁一天"的流动性黑洞

当前应用（2025年）：A股处于"宽货币+信用由紧转松+弱增长修复中"的过渡阶段。这一阶段红利资产（高股息）仍是底仓，但成长股（新质生产力）的配置价值随信用修复而逐步上升。

朱格拉周期（设备投资周期）的核心逻辑：

  1. 经济存在约8-10年的设备更新换代周期
  2. 找到投资结构占比上升或转型方向的行业 → 这类行业具备"更早启动、更强收益、更持久"的特征
  3. 朱格拉受益行业的股价往往提前经济见底约1-2个季度，超额收益可以穿越市场顶部

国金证券的Capex三要素筛选框架：

要素一：充足的扩张空间
  → 行业国产化率低（如半导体设备国产化率约20%）

要素二：充足的扩张能力
  → 行业现金流充裕（经营现金流/营收 > 10%）

要素三：充足的扩张弹性
  → 资本开支周期处于低位（Capex/折旧 < 1，底部特征）
  → 产能利用率开始回升
  → 在手订单/营收比提升

5.4 框架的迭代——旧框架正在失效

东吴证券提出的"框架失效"警告：

变化一：信用周期的波动性收敛

过去（2000-2020）：
  信用扩张/收缩幅度大 → 地产+基建驱动的信用脉冲 → ~3年一轮

现在（2021-至今）：
  经济进入高质量发展 → 政策"托而不举" → 信用扩张幅度收窄
  → 旧框架中"大开大合"的信用周期可能不再重现
  → 基于旧框架的"信用一放就涨"的简单判断将失效

变化二：信用周期与地产的脱钩

过去：社融增速 ≈ 房地产贷款增速（相关系数约0.85）
  地产下行 → 信用收缩 → A股承压 → 逻辑链条清晰

现在：存量房地产贷款占比从2019年末的高点持续下行
  地产不再是信用的唯一引擎
  → 制造业贷款、绿色贷款、科技贷款占比上升
  → 信用结构与A股行业结构同步"脱地产化"

新框架：全球制造业周期+货币政策周期

东吴证券的新研判框架：

全球服务业PMI回落 → 欧美降息周期打开
  → 全球制造业共振补库（库存周期见底）
  → 中国制造业出口景气度提升（PMI新出口订单回升）
  → 中国制造业企业盈利改善
  → A股制造业板块盈利预期上修
  → 整体A股估值中枢上移

关键变量：
  ✓ 全球PMI新订单指数（特别关注欧元区和东南亚）
  ✓ 美国库存周期位置（当前：主动去库末期）
  ✓ 中国PMI新出口订单
  ✓ 航运价格指数（CCFI/SCFI）（反映贸易活跃度）

核心启示：过去看A股要看"国内宏观"（M2、社融、地产投资）；未来需要同时看"全球制造"（全球PMI、库存周期、贸易链）。A股的研判正在从"内循环视角"向"内外双循环视角"升级。

7.2 理解A股的七个核心命题（深度论证版）

命题一：政策是第一驱动力

政策变化直接改变市场的风险溢价（ERP），这是A股区别于成熟市场最根本的特征。

论据：

  • A股年均约12次重大政策调整，远超美股（约3-4次/年）
  • 每一次历史性底部都对应着重大政策信号（2005年股改→998点；2014年国九条+降息→1974点；2018年民企座谈会→2440点；2024年国九条+9月组合拳→2635点）
  • 每一次历史性顶部都伴随着政策收紧（2007年印花税半夜上调→6124点；2015年配资清理→5178点）
  • 政策通过ERP渠道影响的是整个估值中枢，而非个股——这也是A股比美股更容易出现"齐涨齐跌"的制度原因

命题二：散户边际定价，产业资本决定筹码归属

持仓上产业资本（约43%）和散户（约40%）是双主体，但交易上散户占70%+。这意味着短期价格由散户情绪驱动，中长期方向由产业资本行为（增减持、回购、分红）决定。

论据：

  • 小资金（<10万）账户98%亏损 vs 大资金（>1000万）99%盈利——散户在交易层面系统性输给机构和产业资本
  • 产业资本减持潮往往对应市场顶部（2007、2015、2020-2021），增持/回购潮往往对应市场底部（2008、2018、2024）
  • A股最大的"暗线"不是技术分析，不是宏观判断，而是产业资本的增减持方向

命题三："政策底→信用底→市场底→盈利底"是经典传导链

利好政策是"量变"，信用和基本面数据的验证才是"质变"。

论据：

  • 2008年、2014年、2019年、2024年四轮周期的复盘数据支持这一传导链
  • 时滞通常为1-6个月
  • 当前（2025年）处于"市场底→盈利底"的过渡阶段，等待PPI和企业盈利的质变信号

命题四：流动性定价从"海外定价"切换为"国内定价"

过去A股估值受美债利率和利差倒挂严重制约；2024年起国内利率下行和社融修复成为更重要的定价因子。

论据：

  • 2021-2023年中美利差倒挂导致外资流出和估值承压
  • 2024年国内降息力度加大且社融修复，国内因素重夺定价权
  • 这不是短期现象，而是中国货币政策独立性增强的长期趋势

命题五：定价叙事从"增长叙事"转为"竞争力叙事"

市场不再简单看GDP增速，而是看龙头企业在全球关键赛道（AI、半导体、新能源）的竞争胜率——定价的是永续增长预期g*，而非当期盈利。

论据：

  • 宁德时代市值超茅台成为A股市值冠军——资本市场市值龙头从白酒变新能源，是叙事转换的标志性事件
  • 前10%公司贡献A股市值60-70%——市场结构本身就在执行"竞争力筛选"
  • 传统宏观指标（GDP、社会零售、居民收入）对A股的指示意义系统性下降

命题六：制度不对称是"散户98%亏损"的结构性根源

T+1 vs 变相T+0套利、涨跌停聚集效应、做空受限 vs 量化微秒响应——这些机制不对称使散户处于天然劣势。

论据：

  • T+1锁定散户24小时，机构/游资可通过底仓滚动实现变相T+0
  • 涨跌停创造"打板"文化，"打板一时爽，五天火葬场"
  • 量化交易占成交额29%，散户信号延迟至少300ms
  • 这些制度设计初衷是"保护散户"，但实际效果是制造了新的不对称——保护性制度反而保护了专业投资者对散户的收割

命题七：改革方向是"融资市→投资市"，但以十年为单位

注册制解决"入口"问题，退市常态化解决"出口"问题，中长期资金入市改善投资者结构，集体诉讼和严监管保护投资者。

论据：

  • 注册制从试点到全面落地用了5年（2019-2023），但真正"磨合成熟"需要再一个5年
  • 退市率从0.3%提升至1%用了5年，要提升至3-5%还需要更长时间
  • 养老金/长线资金入市从"鼓励"到"建立机制"，但体量达到美股的35%占比需要10-20年
  • "制度建设在加速，但市场生态的质变需要十年为单位的时间"——这是理解A股改革节奏的关键认知

7.5 结语

中国A股是一个只有35年历史的"青年"市场——
它经历过从零到全球第二的飞速扩张，也经历过6124到1664的惨烈暴跌；
它拥有了全球最庞大的散户群体，也承受着由此带来的非理性波动；
它正在经历从"审批制"到"注册制"的制度基因重塑，也正在从"融资市"向"投资市"艰难转身。

理解A股的底层逻辑，不是背公式，而是理解这五个字：
"制度在塑造市场。"

每一次国九条、每一次注册制扩围、每一次退市新规——
都在一点一点地改变这个市场的基因。

而真正的投资机会，
往往就藏在制度变革与市场惯性之间的"认知差"里。
